## Overview

## Quick Start

### Opening Display Settings

1. Open any song
2. Tap the text format icon (ðŸ”¤) in the toolbar

**Fastest Way to Customize**:
1. Open Display Settings
2. Tap "Presets" tab

   - **Stage Performance**: High visibility, bold text, always light mode
   - **Practice**: Comfortable reading with elegant fonts
   - **Large Print**: Maximum readability with accessibility features

## Font Settings

### Base Font Size

**Control**: Slider (12pt - 32pt)

**Purpose**: Sets the primary text size for lyrics

**Recommended Sizes**:
- **12-14pt**: Compact view, more content visible
- **16-18pt**: Standard comfortable reading (default: 16pt)
- **20-24pt**: Stage performance, large print
- **28-32pt**: Maximum accessibility

**Pro Tip**: Chords automatically scale relative to lyrics size

### Font Weight

**Options**:
- **Light**: Elegant, minimal appearance
- **Regular**: Standard weight (default)
- **Medium**: Slightly emphasized
- **Bold**: Maximum visibility and emphasis

**Use Cases**:
- **Light/Regular**: Practice, study sessions
- **Medium**: Rehearsal with moderate lighting
- **Bold**: Stage performance, bright lights

**Note**: Bold text can also be enabled separately in Accessibility settings

### Font Families

Lyra allows you to set different fonts for each text type:

#### Title Font
- **Purpose**: Song titles and headers
- **Options**: System, New York (serif), Monospaced, Georgia, Charter, Century Gothic
- **Default**: System (San Francisco)
- **Recommendation**: System or New York for professional appearance

#### Metadata Font
- **Purpose**: Key, tempo, time signature, artist info
- **Options**: Same as Title
- **Default**: System
- **Recommendation**: System or Charter for subtle elegance

#### Lyrics Font
- **Purpose**: Song lyrics and main content
- **Options**: All fonts including display fonts
- **Default**: System
- **Popular Choices**:
  - **System**: Clean, modern, excellent readability
  - **Charter**: Classic, elegant serif font
  - **Georgia**: Traditional, warm serif
  - **Century Gothic**: Clean, geometric sans-serif

#### Chords Font
- **Purpose**: Chord symbols above lyrics
- **Options**: Emphasizes monospaced options
- **Default**: Monospaced (SF Mono)
- **Recommendation**: Always use monospaced for precise chord alignment
- **Options**:
  - **Monospaced/SF Mono**: Modern, clean (recommended)
  - **Menlo**: Technical, programmer-friendly
  - **Courier**: Classic typewriter look

### Font Preview

Each font section includes a live preview showing:
- Title with selected title font
- Metadata with selected metadata font
- Sample chords with chords font
- Sample lyrics with lyrics font

Test readability before saving!

## Layout Settings

### Line Spacing

**Purpose**: Vertical space between lines of lyrics

**Options**:
- **Compact** (4pt): Dense, more content visible
- **Normal** (8pt): Balanced (default)
- **Relaxed** (12pt): Spacious, easy scanning

**Recommendations**:
- **Compact**: When space is limited
- **Normal**: General use
- **Relaxed**: Stage performance, large print

### Chord-Lyrics Spacing

**Control**: Slider (4pt - 16pt)

**Purpose**: Vertical gap between chords and lyrics below them

**Recommended Settings**:
- **4-6pt**: Tight, compact view
- **8pt**: Standard (default)
- **10-16pt**: Spacious, clear separation

**Pro Tip**: Increase for stage performance to clearly distinguish chords from lyrics

### Section Spacing

**Control**: Slider (16pt - 64pt)

**Purpose**: Vertical space between song sections (verse, chorus, bridge)

**Recommended Settings**:
- **16-24pt**: Compact, less scrolling
- **32pt**: Standard (default)
- **40-64pt**: Clear visual separation

**Use Case**: Large spacing helps quickly identify section breaks during performance

### Margins

**Controls**: Four sliders (0pt - 48pt each)
- **Left Margin**: Space from left edge
- **Right Margin**: Space from right edge
- **Top Margin**: Space from top
- **Bottom Margin**: Space from bottom

**Default**: 20pt all sides

**Recommendations**:
- **Narrow Screens**: 16-20pt
- **Wide Screens**: 24-32pt
- **Stage Performance**: 24pt+ for comfortable reading distance

**Pro Tip**: Adjust left/right margins to control line width for optimal readability

### Two-Column Mode

**Toggle**: On/Off

**Purpose**: Display lyrics in two columns on wide screens (iPads in landscape)

**Benefits**:
- Utilizes wide screen space
- Reduces scrolling
- Easier to see full song at once

**Recommendations**:
- **Enable**: iPad landscape, external displays
- **Disable**: iPhone, iPad portrait, when annotations are heavy

**Note**: Automatically adapts to available width

## Dark Mode Settings

### Dark Mode Preference

**Options**:
- **System**: Follows device settings (default)
- **Always Light**: Forces light mode regardless of system
- **Always Dark**: Forces dark mode regardless of system

**Use Cases**:

#### System
- **Best For**: General use, automatic adaptation
- **Behavior**: Light during day, dark at night (if device configured)
- **Recommendation**: Most users

#### Always Light
- **Best For**: Stage performance, bright environments
- **Reason**: Stage lights wash out dark mode
- **Critical**: Use this for live performance to ensure readability under bright lights
- **Recommendation**: Include in "Stage Performance" preset

#### Always Dark
- **Best For**: Low-light rehearsals, night practice
- **Reason**: Reduces eye strain in dark rooms
- **Recommendation**: Late-night practice sessions

### Dark Mode Previews

The Dark Mode tab includes side-by-side previews:
- **Light Mode Preview**: Shows how colors look in light mode
- **Dark Mode Preview**: Shows how colors look in dark mode

**Important**: Some colors look different in light vs. dark mode. Always check both previews!

## Accessibility Settings

### High Contrast Mode

**Toggle**: On/Off

**Purpose**: Increases color contrast for better readability

**Effects**:
- Enhances color saturation
- Sharpens text edges
- Improves visibility in bright light

**Recommendations**:
- **Enable**: Vision impairments, bright sunlight, stage lights
- **Disable**: Normal indoor lighting

### Reduce Transparency

**Toggle**: On/Off

**Purpose**: Removes transparency effects for clearer backgrounds

**Effects**:
- Solid background colors (no transparency)
- Sharper visual hierarchy
- Less visual noise

**Recommendations**:
- **Enable**: Focus issues, visual sensitivity
- **Disable**: Prefer modern aesthetic

### Bold Text

**Toggle**: On/Off

**Purpose**: Makes all text bold for maximum readability

**Effects**:
- Overrides font weight setting
- All text rendered in bold
- Increases visual weight

**Recommendations**:
- **Enable**: Stage performance, vision impairments, bright environments
- **Disable**: Normal reading, prefer lighter weights

**Note**: Independent of Font Weight settingâ€”works on top of selected weight

### Minimum Font Size

**Control**: Slider (12pt - 24pt)

**Purpose**: Ensures text never goes below minimum size

**Default**: 12pt

**How It Works**:
- Automatically increases text below minimum
- Maintains relative sizing above minimum
- Protects against accidentally-too-small text

**Recommended Settings**:
- **12pt**: Standard (default)
- **16pt**: Comfortable baseline
- **20pt**: Large print baseline
- **24pt**: Maximum accessibility

**Use Case**: Set to 16pt for stage performance to ensure chords never become tiny

### Color Blind Friendly

**Toggle**: On/Off

**Purpose**: Applies color blind friendly color palette

**Effects**:
- Automatically adjusts chord colors
- Uses high-contrast combinations
- Optimized for deuteranopia, protanopia, tritanopia

**Recommendation**: Enable if you have color vision deficiency

**Note**: Works in conjunction with Color Blind Friendly Presets

### Accessibility Preview

The Accessibility tab includes a live preview showing all enabled features:
- Bold text if enabled
- High contrast if enabled
- Transparency settings
- Applied minimum font size

Test before saving!

## Best Practices

### Stage Performance Setup

**Checklist**:
- [ ] Apply "Stage Performance" preset OR:
- [ ] Font size: 20pt minimum
- [ ] Font weight: Bold
- [ ] Chord color: Red or Orange
- [ ] Background: White
- [ ] Dark mode: **Always Light**
- [ ] Bold text: Enabled
- [ ] High contrast: Enabled
- [ ] Test from music stand distance

**Critical**: Always test readability at actual performance distance under stage lights!

### Practice Session Setup

**Recommendations**:
- Comfortable font size (16-18pt)
- Easy-on-eyes colors (Blue chords, Gray/Brown lyrics)
- Normal or relaxed spacing
- System dark mode for automatic adaptation
- Disable accessibility features (unless needed)

### Accessibility Setup

**Maximum Readability**:
- Large font size (24pt+)
- Bold text enabled
- High contrast mode enabled
- Color blind friendly enabled
- Minimum font size: 20pt
- Reduce transparency enabled
- Simple, clear colors (Blue/Black)

### Color Coordination Tips

**Harmonious Combinations**:
- **Professional**: Blue chords + Black lyrics + Gray labels
- **Warm**: Orange chords + Brown lyrics + Brown labels
- **Cool**: Teal chords + Black lyrics + Blue labels
- **Bold**: Red chords + Black lyrics + Orange labels

**Avoid**:
- Low contrast combinations (Gray chords + Gray lyrics)
- Too many bright colors
- Yellow on white (poor visibility)

### Font Pairing Tips

**Professional Combinations**:
- **Modern**: System fonts for all
- **Classic**: New York title + Georgia lyrics
- **Technical**: System title + Menlo chords + Charter lyrics
- **Elegant**: Georgia title + Charter lyrics + SF Mono chords

**Rule**: Chords should always be monospaced for proper alignment

## Settings Persistence

### Song-Specific Settings

**Behavior**: Settings saved per song

**How It Works**:
- Each song can have custom display settings
- Settings persist across app restarts
- Changes only affect current song

**Benefit**: Stage songs can have bold/large text, while practice songs have normal text

### Global Default Settings

**Button**: "Set as Default for All Songs"

**Purpose**: Applies current settings to all future songs

**How It Works**:
1. Configure settings to your preference
2. Tap "Set as Default for All Songs"
3. All new songs use these settings
4. Existing songs keep their custom settings (unless removed)

**Use Case**: Set your preferred general settings once

### Removing Custom Settings

**Button**: "Remove Custom Settings"

**Availability**: Only visible if song has custom settings

**Effect**:
- Removes song-specific settings
- Song reverts to global default
- Cannot undoâ€”must reconfigure if needed

**Use Case**: Reset song to your standard default

### Resetting to Defaults

**Button**: "Reset to Defaults"

**Effect**: Immediately reverts all settings to factory defaults

**Note**: Does not auto-saveâ€”tap "Done" to keep reset settings

## Troubleshooting

### Text Too Small

**Problem**: Can't read text at performance distance

**Solutions**:
- Increase base font size
- Apply "Stage Performance" or "Large Print" preset
- Enable bold text
- Set minimum font size higher

### Colors Look Wrong

**Problem**: Colors don't look as expected

**Possible Causes**:
1. Dark mode active (changes color appearance)
2. Device True Tone enabled
3. Night Shift active

**Solutions**:
- Check both light and dark mode previews
- Disable True Tone temporarily
- Use "Always Light" mode for consistent appearance
- Test in actual performance environment

### Chords Misaligned

**Problem**: Chords don't line up with lyrics

**Cause**: Non-monospaced font selected for chords

**Solution**:
- Set chords font to Monospaced, SF Mono, Menlo, or Courier
- Never use System or display fonts for chords

### Layout Feels Cramped

**Problem**: Too much content, hard to read

**Solutions**:
- Increase line spacing to Relaxed
- Increase section spacing (40-48pt)
- Increase margins (24-32pt)
- Increase font size
- Enable two-column mode (iPad landscape)

### Settings Not Saving

**Problem**: Changes don't persist

**Cause**: Tapped "Cancel" instead of "Done"

**Solution**:
- Always tap "Done" (top right) to save changes
- "Cancel" discards all changes

### Preset Doesn't Look Right

**Problem**: Preset settings don't match expectation

**Cause**: Preset designed for different use case

**Solution**:
- Apply preset as starting point
- Manually adjust specific settings
- Save as global default if desired

## FAQ

**Q: Can I have different settings for different songs?**
A: Yes! Each song can have custom display settings. Configure per song in Display Settings.

**Q: How do I make all my songs use the same settings?**
A: Configure desired settings, tap "Set as Default for All Songs". Future songs will use these settings.

**Q: What's the best preset for stage performance?**
A: "Stage Performance" preset with Always Light mode. Test at actual performance distance!

**Q: Can I use different fonts for chords and lyrics?**
A: Yes! Set separately in Font Families section. Always use monospaced fonts for chords.

**Q: Do settings work with autoscroll?**
A: Yes, all display settings apply during autoscroll.

**Q: Can I share my settings with other musicians?**
A: Not yet, but preset sharing is planned for a future update.

**Q: Why does bold text make everything bold, even if I selected Light weight?**
A: Bold Text (accessibility) overrides font weight for maximum readability. Disable if you want to use selected weight.

**Q: What's the difference between Font Weight and Bold Text?**
A: Font Weight is a style choice (Light/Regular/Medium/Bold). Bold Text is an accessibility override that makes everything bold regardless of weight setting.

**Q: Can I use custom fonts?**
A: Currently limited to system and built-in fonts. Custom font support may be added in future.

**Q: Do color blind friendly features work for all types of color blindness?**
A: Yes, optimized for deuteranopia, protanopia, and tritanopia (the most common types).

**Q: Can I have different settings for light and dark mode?**
A: Not currently. Settings apply to both modes. Use Dark Mode Preference to control which mode is used.

**Q: What's two-column mode good for?**
A: iPad landscape mode where width is much greater than height. Reduces scrolling and shows more content.
