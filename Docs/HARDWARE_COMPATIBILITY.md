## Compatibility Checklist

### Before Purchasing Hardware

**For iPads:**
- [ ] Check iOS version (16.0+ required)
- [ ] Verify storage space (16GB+ recommended)
- [ ] Consider screen size (12.9" best for lyrics)

**For MIDI Devices:**
- [ ] USB class compliant? (works without drivers)
- [ ] Bluetooth MIDI support? (for wireless)
- [ ] Bus-powered or needs external power?
- [ ] iOS compatibility listed by manufacturer?

**For Audio Interfaces:**
- [ ] Core Audio compatible?
- [ ] USB class compliant?
- [ ] Tested on iOS by manufacturer?
- [ ] Buffer size adjustable?

**For Adapters/Hubs:**
- [ ] Official Apple or certified?
- [ ] Supports power pass-through?
- [ ] All needed ports available?
- [ ] Quality brand (Anker, Belkin, etc.)?

## Known Issues

### MIDI Devices

**Roland A-49:** Works but requires firmware update
**Arturia KeyStep:** Works great, no issues
**Novation Launchkey:** Works, but some controls need mapping

### Audio Interfaces

**M-Audio M-Track:** Intermittent connection issues
**Steinberg UR22:** Works but requires specific buffer settings
**MOTU M2:** Works perfectly, highly recommended

### Adapters

**Generic Lightning to HDMI (<$15):** Frequent disconnects
**Cheap USB-C hubs:** Power delivery issues
**Non-certified adapters:** May not pass iPad certification

## Testing Before Performance

### Hardware Checkout Procedure

**1. Physical Connections:**
```
[ ] All cables firmly connected
[ ] No loose connections
[ ] Adapters seated properly
[ ] Power sources verified
```

**2. Device Recognition:**
```
[ ] External display detected
[ ] MIDI devices appear in settings
[ ] Audio interface recognized
[ ] Foot pedals paired
```

**3. Functional Test:**
```
[ ] Display shows correct content
[ ] MIDI controls work
[ ] Audio plays through interface
[ ] Foot pedals trigger actions
```

**4. Stress Test:**
```
[ ] Run for 10 minutes continuous
[ ] Use all features simultaneously
[ ] Monitor for disconnections
[ ] Check performance metrics
```

**5. Backup Verification:**
```
[ ] Backup pedal works
[ ] Manual controls functional
[ ] Printed lyrics available (if needed)
[ ] Alternate iPad ready (if available)
```
