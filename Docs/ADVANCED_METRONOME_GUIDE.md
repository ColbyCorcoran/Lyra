# Advanced Metronome System

Professional metronome with complex time signatures, tempo trainer, and click track generation.

## Overview

Lyra's professional metronome goes far beyond basic click functionality:
- **Complex time signatures**: 5/8, 7/8, 11/8, and more
- **Subdivisions**: 8ths, 16ths, triplets, quintuplets
- **Count-in**: Configurable 1-8 bars with visual/audio cues
- **Tempo trainer**: Gradual tempo increase for practice
- **Click track export**: Generate audio files for DAW use
- **Audio routing**: Click to in-ears, separate from backing
- **Visual metronome**: Multiple display styles
- **Custom accent patterns**: Create any rhythm pattern
- **Mixed meters**: Alternating time signatures
- **Practice session tracking**: Save tempo progression

Perfect for serious practice, complex music, and professional performance.

## Time Signatures

### Common Time Signatures

**4/4 (Common Time)**
```
4 beats per bar, quarter note gets the beat
Most popular time signature
Icon: C
```

**3/4 (Waltz)**
```
3 beats per bar, quarter note gets the beat
Waltz, minuet, some ballads
Accent: Strong-weak-weak
```

**2/2 (Cut Time)**
```
2 beats per bar, half note gets the beat
Marches, fast classical
Icon: Â¢
Feels like fast 4/4
```

### Compound Meters

**6/8**
```
6 beats per bar, eighth note gets the beat
Grouped as 2 groups of 3 (1-2-3, 4-5-6)
Accent: Strong-weak-weak-medium-weak-weak
Common in: Irish jigs, some ballads
```

**9/8**
```
9 beats per bar, eighth note gets the beat
Grouped as 3 groups of 3
Accent: Strong-weak-weak-medium-weak-weak-weak-weak-weak
Common in: Progressive rock, classical
```

**12/8**
```
12 beats per bar, eighth note gets the beat
Grouped as 4 groups of 3
Feels like slow 4/4 with triplet feel
Common in: Blues, gospel, slow ballads
```

### Complex/Odd Meters

**5/8**
```
5 beats per bar, eighth note gets the beat
Grouping options:
- 3+2 (Strong-weak-weak-medium-weak)
- 2+3 (Strong-weak-medium-weak-weak)
Common in: Dave Brubeck, prog rock, Balkan music
```

**7/8**
```
7 beats per bar, eighth note gets the beat
Grouping options:
- 3+2+2
- 2+2+3
- 2+3+2
Common in: Progressive metal, jazz, world music
Pink Floyd "Money" uses 7/4
```

**11/8**
```
11 beats per bar
Grouping: Usually 3+3+3+2 or variations
Very complex, requires practice
Common in: Avant-garde, experimental music
```

**5/4**
```
5 beats per bar, quarter note gets the beat
Grouping: 3+2 or 2+3
Famous: "Take Five" by Dave Brubeck
Common in: Jazz, prog rock
```

### Mixed/Alternating Meters

## Subdivisions

### Quarter Notes (Standard)
```
1 click per beat
Default for most practice
Clear, simple rhythm
```

### 8th Notes
```
2 clicks per beat
Doubles the click rate
Good for: Slow tempos, detailed practice
Example at 60 BPM: Sounds like 120 BPM
```

### 16th Notes
```
4 clicks per beat
Very fast clicking
Good for: Very slow tempos, micro-timing
Example at 40 BPM: Sounds like 160 BPM
```

### Triplets
```
3 clicks per beat
Divides beat into 3 equal parts
Essential for: Shuffle feel, swing, 12/8 time
Sound: "tri-pl-et, tri-pl-et"
```

### Quintuplets
```
5 clicks per beat
Advanced subdivision
Divides beat into 5 equal parts
Common in: Contemporary classical, avant-garde
```

### Sextuplets
```
6 clicks per beat
Very fast subdivision
Two triplets per beat
Common in: Fast classical passages
```

### Mixed Subdivisions

Create custom patterns:
```
Beat 1: Quarter note
Beat 2: Triplet
Beat 3: 8th notes
Beat 4: Quarter note
Perfect for complex rhythms
```

## Troubleshooting

**Click not audible:**
- Check volume slider
- Verify not muted
- Check audio output device

**Tempo seems off:**
- Verify BPM setting
- Check subdivision (16ths sound fast)
- Ensure no tempo trainer active

**Count-in not working:**
- Enable count-in in settings
- Check count-in bars > 0
- Verify audio not muted

**Visual metronome laggy:**
- Reduce visual size
- Close other apps
- Disable complex styles

**Click track export fails:**
- Check available storage
- Reduce duration if very long
- Simplify tempo map

## Best Practices

### Daily Practice

1. **Warm up slow**: Start 20% below target
2. **Build gradually**: Tempo trainer recommended
3. **Perfect first**: Speed comes after accuracy
4. **Take breaks**: Prevents rushing
5. **Track progress**: Save practice sessions

### Live Performance

1. **Test before gig**: Verify click audible
2. **Backup device**: Extra phone/iPad with click
3. **Wired preferred**: Bluetooth can have latency
4. **Volume check**: Click loud enough, not too loud
5. **Practice with click**: Don't use it first time live

### Teaching

1. **Visual + Audio**: Students see and hear
2. **Subdivisions**: Help students feel micro-timing
3. **Slow practice**: Use tempo trainer together
4. **Custom patterns**: Teach complex rhythms
5. **Export clicks**: Give students practice tracks

## FAQ

**Q: What's the tempo range?**
A: 40-300 BPM. Extreme tempos (10-400) available in advanced mode.

**Q: Can I use metronome while backing track plays?**
A: Yes! Independent volume controls for both.

**Q: Does metronome drain battery?**
A: Minimal impact. Audio is efficient. Visual uses more power.

**Q: Can I export click track commercially?**
A: Yes, click tracks you generate are yours to use.

**Q: Metronome latency?**
A: <5ms typical. Wired headphones recommended for critical timing.

**Q: Maximum bars for mixed meter?**
A: No limit, but 4-8 bars practical maximum.

**Q: Can I sync metronome across devices?**
A: Not directly. Use Bluetooth audio for synchronized listening.
