# Accessibility Quick Reference

## Fast Access to Accessibility Features

### Settings Location

**Main Menu:** Settings > Accessibility

### VoiceOver Rotors

**Activate Rotor:**
1. Rotate two fingers on screen
2. Select "Sections" or "Chords"
3. Swipe up/down to navigate

**Available Rotors:**
- **Sections** → Jump between verse, chorus, bridge
- **Chords** → Navigate to each chord in song

### Voice Commands

```
"Hey Siri, play song" → Start autoscroll
"Hey Siri, next song" → Skip forward
"Hey Siri, transpose up" → Transpose +1
"Hey Siri, start metronome" → Begin click
```

### Switch Control Speeds

**Recommended Settings:**
- Beginners: 2.0s - 3.0s
- Intermediate: 1.0s - 2.0s
- Advanced: 0.5s - 1.0s

### High Contrast Modes

| Mode | Contrast Ratio | Best For |
|------|---------------|----------|
| Off | 1.0x | Normal vision |
| Standard | 1.5x | Mild low vision |
| Ultra | 2.0x | Severe low vision |

### Font Size Multipliers

| Setting | Use Case |
|---------|----------|
| 0.8x | Compact mode, fit more on screen |
| 1.0x | Default, readable |
| 1.5x | Large, easier reading |
| 2.0x | Extra large, low vision |

### Haptic Strength

| Level | Intensity | Battery Impact |
|-------|-----------|----------------|
| Off | None | None |
| Light | Subtle | Minimal |
| Medium | Standard | Low |
| Strong | Maximum | Moderate |

### Animation Speed

| Speed | Description |
|-------|-------------|
| 0% | Instant (no animation) |
| 25% | Very fast |
| 50% | Fast |
| 75% | Slow |
| 100% | Normal (default) |

### Keyboard Shortcuts

**Song View:**
- `Cmd+E` → Edit
- `Cmd+T` → Transpose
- `Space` → Toggle autoscroll
- `Cmd+M` → Toggle metronome

**Navigation:**
- `Cmd+1` → Library
- `Cmd+2` → Books
- `Cmd+3` → Sets
- `Cmd+,` → Settings

### Quick Troubleshooting

**VoiceOver not reading chords:**
→ Settings > Accessibility > VoiceOver > Describe Chord Diagrams

**Buttons too small:**
→ Settings > Accessibility > Large Buttons Mode

**Can't see text:**
→ Settings > Accessibility > High Contrast Mode > Ultra

**Animations causing discomfort:**
→ iOS Settings > Accessibility > Reduce Motion

**Need simpler interface:**
→ Settings > Accessibility > Simplified Mode

### Emergency Reset

If accessibility settings cause issues:

1. Go to Settings > Accessibility
2. Scroll to bottom
3. Tap "Reset to Defaults"
4. Confirm reset

All settings return to safe defaults.

### Get Help
