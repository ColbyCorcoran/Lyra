- **Scroll Position**: Heel = Top, Toe = Bottom
- **Autoscroll Speed**: Adjust tempo real-time
- **Volume/Brightness**: Dynamic control

## Multi-Pedal Support

Connect up to 4 pedal devices simultaneously!

**Chain Modes:**
- **Independent**: Each device separate mapping
- **Sequential**: All pedals numbered 1-12
- **Mirrored**: All pedals same actions

## Troubleshooting Fast Fixes

| Problem | Solution |
|---------|----------|
| Not detected | Re-pair Bluetooth, check battery |
| Wrong action | Verify active profile in Test Mode |
| Double-press fails | Increase timing window (Settings) |
| Expression jumpy | Increase smoothing to 50-80% |
| Delayed response | Reduce smoothing, check battery |

## Pro Tips

1. **Test before every gig** - Run Test Mode
2. **Keep it simple** - Fewer actions = better muscle memory
3. **Label your pedals** - Tape on floor for dark stages
4. **Bring spare batteries** - Always
5. **Use long-press for emergencies** - Blank, jump, reset
6. **Context-aware mode** - Let Lyra auto-switch modes
7. **Network sync for bands** - Leader controls all charts
8. **Expression for dynamics** - Natural performance feel
