10. **Sound check**: Test audio routing early

## Example Workflows

### Solo Acoustic with Backing
```
1. Import full backing track MP3
2. Enable sync autoscroll
3. Assign play/pause to foot pedal
4. Adjust volume for venue
5. Add markers for key changes
6. Perform!
```

### Band with Separate Click
```
1. Import backing to Track 1
2. Import click to Track 2
3. Route click to in-ears (headphones)
4. Route backing to PA (main output)
5. Mixer: Click at 50%, Backing at 80%
6. EQ backing to cut through live sound
7. Ready!
```

### Worship Leader Multi-Track
```
1. Track 1: Full band (75% volume)
2. Track 2: Click (headphones only)
3. Track 3: Choir (60% volume)
4. Track 4: Strings (40% volume)
5. Pan choir slightly left
6. Pan strings slightly right
7. Master volume at 80%
8. Solo each track to verify
9. Perform with full control!
```
