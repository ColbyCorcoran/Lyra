# Advanced Metronome - Quick Reference

## Quick Start

**Basic:** Tap metronome icon ‚Üí Set BPM ‚Üí Play

## Common Time Signatures

| Signature | Name | Beats | Use |
|-----------|------|-------|-----|
| 4/4 | Common Time | 4 quarter notes | Most pop/rock |
| 3/4 | Waltz | 3 quarter notes | Waltz, ballads |
| 6/8 | Compound | 6 eighth notes (2x3) | Jigs, some ballads |
| 2/2 | Cut Time | 2 half notes | Marches |
| 5/8 | Complex | 5 eighths (3+2 or 2+3) | Prog rock, jazz |
| 7/8 | Complex | 7 eighths (3+2+2) | Prog, world music |
| 12/8 | Compound | 12 eighths (4x3) | Blues, gospel |

| **Sextuplet** | 6 | Fast classical |

## Accent Patterns

- **Downbeat**: Strong-weak-weak-weak (most common)
- **Backbeat**: Weak-strong-weak-strong (rock/pop)
- **Waltz**: Strong-weak-weak (3/4 time)
- **Every 2**: Strong-weak (alternating)
- **Custom**: Create any pattern

## Tempo Trainer

**Purpose:** Gradually increase tempo for practice

**Setup:**
```
Start: 60 BPM (where you can play it)
Target: 120 BPM (goal)
Increment: 5 BPM (increase by)
Interval: 4 bars (every)
```

**Modes:**
- **Auto**: Increases automatically
- **Manual**: You control progression
- **Random**: Random tempo changes (builds adaptability)

**Track Progress:** Sessions automatically saved

## Common Workflows

### Building Speed (Solo Practice)
```
Week 1: 80 BPM
Week 2: 90 BPM
Week 3: 100 BPM
Week 4: 110 BPM
Target: 120 BPM achieved!
```

### Learning Complex Meter (7/8)
```
1. Start: 40 BPM, subdivision: 8ths
2. Accent: 3+2+2 pattern
3. Clap pattern first
4. Add instrument at 50 BPM
5. Increase by 5 BPM weekly
6. Target: 120 BPM
```

### Recording with Click
```
1. Export click track
2. Import to DAW
3. Record to click
4. Mute click in mix
```

### Band Setup
```
Click ‚Üí In-ear monitors (headphones)
Backing ‚Üí PA system

```

## Time Signature Cheat Sheet

**Simple Meters** (2, 3, 4 beats):
```
2/4: March (Strong-weak)
3/4: Waltz (Strong-weak-weak)
4/4: Common (Strong-weak-medium-weak)
```

**Compound Meters** (6, 9, 12 beats, grouped by 3):
```
6/8: 1-2-3-4-5-6 (feels like 2 groups)
9/8: 1-2-3-4-5-6-7-8-9 (feels like 3 groups)
12/8: Like slow 4/4 with triplet feel
```

**Complex Meters** (5, 7, 11 beats):
```
5/8: 1-2-3-4-5 (group as 3+2 or 2+3)
7/8: 1-2-3-4-5-6-7 (group as 3+2+2 or 2+2+3)
11/8: Usually 3+3+3+2 or variations
```

## Accent Sound vs Regular

**Best combinations:**

Regular ‚Üí Accent:
- Click ‚Üí Woodblock (classic)
- Beep ‚Üí Beep (louder)
- Hi-Hat ‚Üí Rim Shot (drums)
- Clave ‚Üí Cowbell (Latin)

**Volume ratio:**
- Regular: 70%
- Accent: 90%
- Difference helps distinguish beats
