# Stage Monitors - Quick Start Guide

Get your stage monitors set up in 5 minutes!

## Setup Steps

### 1. Choose Your Setup Type

**Option A: Physical Displays** (most common)
- Connect external displays via HDMI/USB-C
- Lyra detects them automatically
- Best for fixed stage setups

**Option B: Network Monitors** (wireless)
- Use iPads/iPhones as monitors
- Connect via WiFi
- Best for flexible setups or large stages

**Option C: Hybrid** (both physical and wireless)
- Combine both approaches
- Maximum flexibility

### 2. Quick Setup (Physical Displays)

1. Open Lyra on your main device
2. Go to **Settings** ‚Üí **Stage Monitors**
3. Tap **"Small Band"**, **"Full Band"**, or **"Worship Team"** preset
4. Connect your displays
5. Done! Monitors auto-configure

### 3. Quick Setup (Network Monitors)

**On Leader Device:**
1. Settings ‚Üí Stage Monitors ‚Üí Network
2. Toggle **"Enable Network Monitors"**
3. Choose **"Auto-Discovery"** mode
4. Note the device name shown

**On Each Monitor Device:**
1. Open Lyra
2. Stage Monitor ‚Üí Join Session
3. Select leader device
4. Choose your role (Vocalist, Lead, etc.)
5. Done!

### 4. Test Your Setup

1. Load a song
2. Tap **Stage Monitor Control**
3. Verify all monitors showing content
4. Use **Next/Previous** buttons to test navigation
5. Adjust font sizes if needed (Settings ‚Üí Stage Monitors ‚Üí tap monitor)

## Common Configurations

### Solo Musician
- 1 external display showing chords + lyrics
- Use rehearsal profile

### Duo/Trio
- Monitor 1: Vocalist (lyrics only)
- Monitor 2: Instrumentalist (chords only)

### Small Band (4-5 members)
- Monitor 1: Vocalist (lyrics)
- Monitor 2: Lead Guitar (chords)
- Monitor 3: Bass (chords)
- Monitor 4: Drummer (song structure)

### Worship Team with Audience
- Monitor 1: Vocals (lyrics)
- Monitor 2: Keys (chords + lyrics)
- Monitor 3: Lead (chords)
- Monitor 4: Audience Display (large lyrics)

## Essential Controls

### During Performance

**Advance Song:**
- Stage Monitor Control ‚Üí **Next** button
- All monitors update automatically

**Blank Monitors:**
- Stage Monitor Control ‚Üí **Blank All**
- Use during breaks or transitions

**Emergency Reset:**
- Stage Monitor Control ‚Üí Tap monitor ‚Üí **Reset to Default**

### Quick Adjustments

**Text Too Small:**
- Settings ‚Üí Stage Monitors ‚Üí Tap monitor
- Adjust "Font Size" slider
- Test from stage position

**Wrong Layout:**
- Settings ‚Üí Stage Monitors ‚Üí Tap monitor
- Change "Layout" picker
- Choose layout that fits your role

**Colors Not Visible:**
- Settings ‚Üí Stage Monitors ‚Üí Tap monitor
- Enable "High Contrast"
- Or choose custom colors

## Troubleshooting in 30 Seconds

**Monitor blank/not showing:**
1. Check if "Blank All" is active (unblank it)
2. Verify display is connected
3. Load a song if none loaded

**Network device not connecting:**
1. Confirm same WiFi network
2. Restart network (Settings ‚Üí Stage Monitors ‚Üí Network ‚Üí toggle off/on)
3. Try manual connection

**Lag/delay on network monitors:**
1. Move closer to WiFi router
2. Settings ‚Üí Stage Monitors ‚Üí Network Settings
3. Increase "Max Latency" to 200ms

**Changes not saving:**
1. Make sure setup has a name
2. Tap "Save" after changes
3. Restart app if issue persists

## Pro Tips

1. **Save Your Setup**: After configuring, tap "Save Setup" with a memorable name
2. **Test Before Performance**: Run through entire set with monitors active
3. **Backup Plan**: Keep printed lyrics/chords just in case
4. **Stage Lighting**: Adjust colors/contrast based on stage lighting during sound check
5. **Font Sizes**: Bigger than you think! Test from actual stage position
